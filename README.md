PageRank Algorithm Project
Overview

This project explores the PageRank algorithm used to rank nodes in a network.
The algorithm was originally developed by Google to rank web pages.

The project includes:
- mathematical explanation of PageRank
- Python implementation
- visualization of results
- experiments on random graphs

Technologies Used:
- Python
- NumPy
- NetworkX
- Matplotlib
- Jupyter Notebook
- Author
